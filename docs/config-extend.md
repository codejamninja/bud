# bud.use

Registers an `array` of [Bud extensions](guide-using-extensions.md) with the framework.

## Usage

```js
bud.extend([require('@roots/bud-sass'), require('@roots/bud-tailwind')])
```

## Arguments

| Name      | Type  |
| --------- | ----- |
| `plugins` | array |
