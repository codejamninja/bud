# bud.distPath

Sets the directory containing a project's source files and assets.

Once set, this directory may be utilitized using [bud.dist](config-dist.md).

## Usage

```js
bud.distPath('public')
```

## Arguments

| Name   | Type   |
| ------ | ------ |
| `path` | string |
